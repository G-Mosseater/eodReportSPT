##  End of Day Reporting Platform

A simple internal tool built to help manage daily operations for a whale watching company.

It creates and tracks end-of-day reports that include everything that happens during the day, such as departures, cancellations, staff involved, and payments. Admin users can fully manage the data, including editing and deleting reports when needed.

The platform also includes:
- Filtering reports by month and year
- Analytics based on tours and dates
- A clear overview of daily operations in one place

More features like private bookings and a live departure screen are currently being developed.

This project was created to replace outdated Excel workflows and make daily operations easier, faster, and more organized.

---

##  Tech Stack

- Next.js  
- TypeScript  
- MongoDB  
- Tailwind CSS  

The design is intentionally minimal, as the main focus was on functionality, reliability, and improving real-world workflows.
